# DX Icon Theme

DX is a theme based on the [Dex KDE icon theme](https://github.com/ishovkun/Dex) and [Papirus](https://github.com/PapirusDevelopmentTeam/papirus-icon-theme)

## :open_file_folder: Installing

1. Download the zip file
2. Extract it
3. Copy the folder and place it in `/usr/share/icons/` or `$HOME/.icons`(create the directory if it does not exist)  .
4. Change the theme with the appearance settings of your Desktop Enviroment (For example Gnome Tweak Tool for GNOME)
